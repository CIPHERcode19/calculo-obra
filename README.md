# Cálculo Obra

Ferramenta web simples para cálculo rápido de quantitativos de betão e aço,
pensada para uso em campo por fiscais de obra e técnicos de construção.

## Funcionalidades

- **Betão**: cálculo de volume por elemento (sapata, pilar, viga, laje, muro),
  com percentagem de desperdício ajustável.
- **Aço**: cálculo de peso de varões por diâmetro (Ø6 a Ø32mm), com base em
  tabela de referência de massa por metro.
- **Resumo**: totais gerais e exportação em texto para relatórios.

## Como usar

Acesse o link do site (GitHub Pages) e comece a lançar os elementos da obra.
Não precisa de instalação nem internet depois de a página carregar.

**Nota:** os cálculos são estimativas de campo para orçamento rápido.
Não substituem o projeto de estabilidade nem a ficha técnica do fabricante.

## Tecnologia

HTML, CSS e JavaScript puro — sem dependências externas.
